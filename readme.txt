OVERVIEW
--------
This project is an end-to-end data engineering pipeline that:
  1. Fetches country and capital data from the REST Countries API
  2. Fetches 7-day weather forecasts from the Open-Meteo API
  3. Cleans and stores the data in a CSV file
  4. Displays the data in an interactive Dash dashboard

No API keys are required. Both APIs are free and public.


PROJECT FILES
-------------
  forecast.py        — fetches data from both APIs and saves to CSV
  dashboard.py       — loads the CSV and runs the interactive dashboard
  forecast_data.csv  — the cleaned dataset (auto-generated by forecast.py)
  README.txt         — this file


REQUIREMENTS
------------
Python 3.x must be installed along with the following libraries:
  - dash
  - plotly
  - pandas
  - requests

To install all required libraries, open Command Prompt and run:

  pip install dash plotly pandas requests


HOW TO RUN
----------
Follow these steps in order every time you want to use the dashboard:

  STEP 1 — Open Command Prompt or VS Code terminal

  STEP 2 — Navigate to the project folder:
    cd "C:\Vineeth\Studies\692 Pipelines to Insights\climate dashboard"

  STEP 3 — Fetch fresh data from the APIs:
    python forecast.py

    Wait until you see: "Done! forecast_data.csv has been saved."

  STEP 4 — Launch the dashboard:
    python dashboard.py

    Wait until you see: "Dash is running on http://127.0.0.1:8050/"

  STEP 5 — Open your browser and go to:
    http://127.0.0.1:8050


DASHBOARD FEATURES
------------------
  - World map        : shows all 10 capital cities as clickable markers
  - Region filter    : filter cities by continent/region
  - Summary card     : click a city to see today's full weather profile
  - Bar chart        : compare high and low temperatures across all cities
  - Forecast table   : 7-day detailed forecast for the selected city
                       (color-coded by travel recommendation and climate risk)


DATA FIELDS
-----------
Each city provides the following data for 7 days:
  - Date
  - High and low temperature (°C)
  - Feels like temperature (°C)
  - Rain probability (%)
  - Weather condition (e.g. Clear sky, Light rain)
  - Day length (hours and minutes)
  - Current season
  - Climate risk flag
  - Travel recommendation
  - Region, languages spoken, and timezone
