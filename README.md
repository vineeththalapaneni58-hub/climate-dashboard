# World Capitals Climate Dashboard

## Project Overview
An end to end ETL pipeline that fetches geographic and weather data
for 10 world capitals, transforms and validates it, loads it into a
SQLite database, and displays it in an interactive Dash dashboard.

## Project Structure
climate dashboard/
pipeline/
extract.py       Fetches data from REST Countries and Open-Meteo APIs
transform.py     Cleans and transforms raw JSON into structured data
load.py          Loads data into SQLite using SQLAlchemy
validate.py      Runs 5 data quality validation checks
data/
forecast_data.csv  Cleaned dataset generated by the pipeline
logs/
pipeline.log     Auto generated log file from each pipeline run
dashboard.py       Interactive Dash dashboard
etl_pipeline.py    Main pipeline runner
